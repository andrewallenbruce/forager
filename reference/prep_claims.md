# Prep mock coding/billing data frame

Prep mock coding/billing data frame

## Usage

``` r
prep_claims(df, ...)
```

## Arguments

- df:

  `[data.frame]` data frame generated by
  [`mock_claims()`](https://andrewallenbruce.github.io/forager/reference/mock_claims.md)

- ...:

  `[dots]` additional arguments

## Value

A [tibble](https://tibble.tidyverse.org/reference/tibble-package.html)

## Examples

``` r
prep_claims(mock_claims(rows = 5))
#> Error in loadNamespace(x): there is no package called ‘fastplyr’
```
